# AI QA Assistant

An AI-powered QA Assistant that generates software test cases from requirements using Gemini AI.

## Features

- Functional Test Case Generation
- Negative Test Case Generation
- Boundary Test Case Generation
- Edge Case Generation
- Streamlit UI

## Tech Stack

- Python
- Gemini AI
- Streamlit
- JSON

## Current Status

Under Development

## How to Run

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
